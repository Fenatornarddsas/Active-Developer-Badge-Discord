# Active Developer Badge Discord
 Built on python, used to easily get active developer badge on discord
 Go to https://discord.com/developers/applications in order to make a bot  (you can watch tutorials on how to make it)
 In Bot tab press reset token and copy it (make sure to save it in notepad)
 Then open the .py file and paste it in
